Aethonan 0.1.0

This bot can notify you about builds from the continuous integration system.
Aethonan on Telegram
Travis CI
A hosted, distributed continuous integration service used to build and test software projects hosted at GitHub.

You can define webhooks to be notified on Telegram about build results, or multiple URLs for multiple groups, as with other notifications types you can specify when webhook payloads will be sent.

Web Site Documentation
Circle CI
Gives software developers powerful Continuous Integration and Deployment with easy setup and maintenance.

This integration will post notifications to a private chat or group in Telegram when a CircleCI build finishes. You can configure per branch build notification. Alternatively, you can only send notifications for branches which match a whitelist. You can combine them, in which case only branch names which do match the whitelist and do not match the blacklist get notifications.
